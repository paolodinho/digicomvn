# Context

Thư mục này chứa **tài liệu tham khảo** cho dự án mà Claude có thể đọc khi cần.

## Gợi ý file nên có (tuỳ dự án)
- `brand.md` — brand voice, tone, guidelines
- `audience.md` — chân dung khách hàng / đối tượng
- `competitors.md` — đối thủ / dự án tham khảo
- `glossary.md` — thuật ngữ riêng
- `references/` — link + trích dẫn nguồn quan trọng
- `examples/` — output mẫu tốt để Claude học theo
