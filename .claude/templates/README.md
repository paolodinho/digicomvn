# Templates

Thư mục chứa **mẫu có sẵn** để Claude dùng lại — giảm thời gian làm việc, tăng nhất quán.

## Gợi ý template nên có
- `email.md` — mẫu email các loại
- `report.md` — mẫu báo cáo
- `meeting-notes.md` — mẫu ghi chú họp
- `brief.md` — mẫu brief khách hàng
- `article.md` — mẫu bài viết
- `component.md` / `endpoint.md` — mẫu code (nếu là dự án code)
